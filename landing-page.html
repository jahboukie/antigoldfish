Why AGM

  • Zero‑trust by default: allow‑listed commands/paths, audit log, receipts, and journal.
  • Machine‑bound encryption‑at‑rest (optional secure mode).
  • Code‑aware indexing: file and symbol chunking; digest cache; rename optimization.
  • Fast local search: SQLite FTS5 with optional hybrid vector rerank (sqlite‑vss or local fallback).
  • Air‑gapped exports: portable .agmctx bundles with optional ED25519 signatures.
  • VS Code tasks and keybindings; watch mode for incremental indexing.
  • No subscriptions, no trials, no telemetry.

Try It

  1. Install: npm i -g antigoldfishmode or clone and npm run build.
  2. Initialize: agm init (adds .antigoldfishmode/ and VS Code tasks).
  3. Prove offline: agm prove-offline --json.
  4. Index: agm index-code --path src --symbols or run watch: agm watch-code --path src.
  5. Search: agm search-code "handleSearch" --hybrid --preview 3.

Privacy

AGM blocks network egress by default for core commands. Export/import memories air‑gapped and verify operations with receipt files and a structured journal. Use agm vector-status and agm health for observability.
